# nest-chat using socket

## start nest.js project

```bash
npm i -g @nestjs/cli
nest new nest-chat
```
---

## start nest.js with typescript

```bash
git clone https://github.com/nestjs/typescript-starter.git nest-chat
cd nest-chat
npm install
npm run start
```
